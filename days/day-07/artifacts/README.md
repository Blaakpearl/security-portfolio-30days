## How to Import the ATT&CK Navigator Layer

1. Open https://mitre-attack.github.io/attack-navigator/
2. Click **"Open Existing Layer"** → **"Upload from local"**
3. Select `navigator_layer_week1.json`
4. The layer loads with all 38 Week 1 techniques colour-coded by day source
